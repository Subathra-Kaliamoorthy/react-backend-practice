# 🔗 React Backend Practice

A practice project built using React to understand backend integration, API communication, and data handling between frontend and backend applications.

---

## 🚀 Features

* Connect React frontend with backend APIs
* Fetch and display data dynamically
* Handle HTTP requests and responses
* Manage asynchronous operations
* Simple and responsive UI

---

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6)
* REST API
* HTML5
* CSS3

---

## 📌 Concepts Implemented

* Backend integration in React
* Fetching data from APIs
* Handling asynchronous operations
* Managing server responses
* Frontend and backend communication

---

## 📌 How to Run

### Frontend Setup

1. Clone the repository:

```bash id="r2m6hf"
git clone https://github.com/Subathra-Kaliamoorthy/react-backend-practice
```

2. Navigate to the frontend project folder:

```bash id="v8p1xd"
cd react-backend-practice
```

3. Install dependencies:

```bash id="x3n9ka"
npm install
```

4. Start the React app:

```bash id="q7w5lt"
npm run dev
```

---

### Backend Setup

1. Navigate to the backend folder:

```bash id="m5z8rv"
cd backend
```

2. Install backend dependencies:

```bash id="d4k2hs"
npm install
```

3. Start the backend server:

```bash id="f7x1qp"
node app.js
```
